# AI Content Detector - Final Year Project

Hey everyone! This is my final year project for detecting whether an image or video is real or generated by AI. It uses the Gemini API under the hood to analyze media files.

## Features
- Detect AI-generated images and videos 
- Keeps track of your analysis history (works offline!)
- Modern UI built with React and Tailwind

## How to run locally

Make sure you have Node.js installed.

1. Clone or download this repo.
2. Run `npm install` to get the packages.
3. You'll need a `.env` file with your `GEMINI_API_KEY`. (There's an `.env.example` to copy from).
4. Run `npm run dev` to start the local server.

That's it! Let me know if you run into any weird bugs.
